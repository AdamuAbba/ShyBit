# ShyBit-CLI

## Table of content

- Requirements
- Installation

### Requirements

- Bitcoin core (installed globally)
- node (installed)
- Internet connection (for api calls)

### installation

### clone this repo

```zsh

git clone https://github.com/AdamuAbba/ShyBit-CLI.git
```

### change directory and install npm packages

```zsh

cd ShyBit-CLI
```

```zsh

npm install
```

### build and run app

```zsh

npm run dev
```
